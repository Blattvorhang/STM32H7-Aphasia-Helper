# STM32H7 Aphasia Helper
失语症患者辅助设备

## Hardware
- STM32H7A3ZI-Q
- OV7670 (Camera): 320x240, 30fps, RGB565
- ST7735 (2.8" TFT LCD): 240x320, RGB565
- ATK-MPU6050 (Gyroscope)
- LD3320 (Voice Recognition)
- Passive Buzzer
